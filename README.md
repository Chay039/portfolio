# Full-Stack Professional Portfolio

A high-performance, responsive portfolio showcasing modern web development techniques, 3D visualizations, and robust backend integrations.

## 🚀 Tech Stack

- **Frontend:** Next.js 14, React, TypeScript
- **Styling:** Tailwind CSS, Framer Motion
- **3D Graphics:** React Three Fiber, Three.js
- **Deployment:** Cloudflare Pages / Workers
- **Authentication & Forms:** Integrated with secure environment variables

## ✨ Key Features

- **Automated CI/CD:** Integrated with Cloudflare for instant deployments on every push.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **Dynamic Contact Form:** Secure server-side email handling using custom Workers.
- **Performance Optimized:** High Lighthouse scores through Next.js image optimization and edge hosting.
